## Exercise 5

Given the following code, what is logged out (try first without coding it):

```js
let sum = 35;
if (sum % 10 == 0){
    console.log("a");
} else if (sum % 2 == 1){
    if (sum % 5 == 0 && sum % 2 == 0){
        console.log("b");
    } else if (sum % 5 == 0){
        console.log("c")
    } else {
        console.log("d")
    }
} else {
    console.log("e")
}
```

What values does `sum`need to log out 'a', 'b', 'c', 'd' and 'e', respectively?
